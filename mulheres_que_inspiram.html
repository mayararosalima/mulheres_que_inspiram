Ada Lovelace

-> Teve a ideia do primeiro programa computacional;
-> Analisou e traduziu diversos materiais dos matemáticos contemporâneos, ajudando para que o primeiro algoritmo do mundo fosse desenvolvido;
-> É considerada a mãe da computação.

Carol Shaw

-> É considerada a primeira mulher que começou a trabalhar com o desenvolvimento de jogos digitais;
-> Criou softwares para games e consoles, sendo pioneira na geração procedural de conteúdo;
-> Foi uma das primeiras colaboradoras da Atari.

Grace Hopper

-> O termo "bug" passou a ser usado depois que Grace Hopper solucionou um erro de processamento de dados ao retirar um inseto (chamado de “bug” na língua inglesa) de dentro de uma máquina;
-> Foi a primeira mulher a ter um PhD em Matemática na universidade de Yale;
-> Trabalhou também na área da tecnologia dentro da Marinha dos Estados Unidos;
-> Foi uma das idealizadoras de uma das linguagens de programação mais utilizadas para banco de dados de negócios, a COBOL.
